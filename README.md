# TwoFa

This project demo-straights a simple Two Factor authentication Angular application. Re-engineered from a React application as a training exercise.

Generated with [Angular CLI](https://github.com/angular/angular-cli) version 8.3.17.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

## API Dependancy

Requires json-server running as follows:

    > json-server db.json --middlewares sca-cwm-api.js
